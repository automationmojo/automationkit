load_from_config(lscape, upnp_coord=None)[source]¶

Loads the SshPoolCoordinator connection info from the config file. The optional upnp_coord parameter is provided when the config file contains UPNP devices that support SSH. The UpnpCoordinator passed as the upnp_coord must be started up so it will have the opportunity to resolve the IP address of the upnp devices.

verify_connectivity(cmd="echo 'It Works'", user=None, raiseerror=True)[source]¶

Loops through the nodes in the SSH pool and utilizes the credentials for the specified user in order to verify connectivity with the remote node.

Parameters

  • cmd (str) – A command to run on the remote machine in order to verify that ssh connectivity can be establish.

  • user (str) – The name of the user credentials to use for connectivity. If the ‘user’ parameter is not provided, then the credentials of the default or priviledged user will be used.
