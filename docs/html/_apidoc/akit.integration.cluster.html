classmethod diagnostic(diag_level: int, diag_folder: str)[source]¶

The API is called by the akit.sequencer.Sequencer object when the automation sequencer is building out a diagnostic package at a diagnostic point in the automation sequence. Example diagnostic points are:

  • pre-run

  • post-run

Each diagnostic package has its own storage location so derived akit.scope.ScopeMixIn objects can simply write to their specified output folder.

Parameters

  • diag_level – The maximum diagnostic level to run dianostics for.

  • diag_folder – The output folder path where the diagnostic information should be written.
