akit.testing.utilities.find_included_modules_under_root(root: str, package: Optional[str], module: str, excluded_path_prefixes: list = [])[source]

Walks through a directory tree starting at a root directory and finds all of the files that corresponded to the package, module expressions specified.

Parameters

  • root – The root directory to start from when performing the tree walk to look for included tests.

  • package – The package name component if there is one. The package components are the directories with __init__.py files up to the file where the module file itself is found. It could be that there is only a module name.

  • module – The module name component. There must be a module because that is the file where the tests are found.
