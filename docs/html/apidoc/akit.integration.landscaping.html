class akit.integration.landscaping.landscapedeviceextension.LandscapeDeviceExtension[source]

Bases: object

The LandscapeDeviceExtension object is the base class object that allows for the extension of functionality for a LandscapeDevice. The LandscapeDeviceExtension is used by the UPnP, SSH and Muse coordinators to extend landscape devices with metadata and device connectivity for the UPnp, SSH and Muse protocols.

property basedevice

Returns a reference to the base LandscapeDevice that this extension was attached to.

property configuration

The protocol specific configuration infor for the device.

property coordinator

Returns a reference to the coordinator that created the LandscapeDeviceExtension.

property extid

A unique device identifier created by the coordinator for this device extension. The identier is typically something associated with the protocol.
