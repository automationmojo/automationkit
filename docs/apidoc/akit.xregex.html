class akit.xregex.regexreader.RegExReader(content: Union[List, str], strict=False)[source]

Bases: object

The RegExReader is a class that makes it easy to create text content parsers.

The user of this class will inherit from the class and override the EXPECTED_LINES class variable and the _process_matches method to capture value matches.

I The user might also want to override the _register_pattern_match_bugs method in order to integrate with custom enterprise but filing functionality for filing bugs for pattern match misses.
