akit.testing.verification.dictionary.assert_api_dict_response_has_keys(api: str, to_inspect: dict, expected_keys: List[str]) → Optional[akit.exceptions.AKitAssertionError][source]

Verifies that the speicified return result from the specified API has the specified expected keys. If the verification fails then an :class::AKitAssertionError is created and return, otherwise None is returned. It is the resposibility of the calling test to raise the returned error.

Parameters

  • api – The name of the API that returned the result being inspected.

  • to_inspect – The dictionary being inspected

  • expected_keys – The list of expected keys

Returns

None or an :class::AKitAssertionError for the caller to raise.
