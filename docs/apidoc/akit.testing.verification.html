akit.testing.verification.assert_dict_response_has_keys(to_inspect: dict, expected_keys: List[str], api: Optional[str] = None)[source]

Verifies that the specified return result from the specified API has the specified expected keys. If the verification fails then an AKitAssertionError is created and return, otherwise None is returned. It is the resposibility of the calling test to raise the returned error.

Parameters

  • to_inspect – The dictionary being inspected

  • expected_keys – The list of expected keys

  • api – The optional name of the API that returned the result being inspected.

Returns

None or an AKitAssertionError for the caller to raise.
