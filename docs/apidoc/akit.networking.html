akit.networking.interfaces.get_correspondance_interface(ref_ip: str, ref_port: int, addr_family=AddressFamily.AF_INET) → Tuple[str, str][source]

Utilizes the TCP stack to make a connection to a remote computer and utilizes gets the network interface that was used to connect to the remote computer. This network interface is the network interface that is likely to be visible to the remote computer and thus could be used to establish services that will be visible to the remote computer.

Parameters

  • ref_ip – An IP address of a computer that is on the subnet that you wish to find the correspondance ip address for and that is hosting a service that will accept a TCP connection from a client.

  • ref_port – The port number of a service on a computer that will accept a TCP connection so we can determine a path to the computer.

  • addr_family – The socket address family to utilize when making a remote connection to a host socket.AF_INET or socket.AF_INET6. The address family used will determine the type of IP address returned from this function.

Returns

The correspondance interface and IPAddress that can be used to setup a service that is visible to the reference IP address.

akit.networking.multicast module

akit.networking.multicast.create_multicast_socket_for_iface(multicast_addr: str, ifname: str, port: int, family: socket.AddressFamily = AddressFamily.AF_INET, ttl: Optional[int] = None, loop: Optional[int] = None, timeout: Optional[float] = None, apple_p2p: bool = False) → socket.socket[source]

Create a socket for listening for multi-cast packets sent to the specified group on the specified interface and with the specified settings.

Parameters

  • multicast_addr – The multicast group address the socket will be used to communicate with.

  • ifname – The name if the interface to create the socket for.

  • port – The port to bind the socket with.

  • family – The internet address family for the socket, either (socket.AF_INET or socket.AF_INET6)

  • ttl – The time to live that will be attached to the packets sent by this socket. 0 = same host 1 = same subnet 32 = same site 64 = same region 128 = same continent 255 = unrestricted scope

  • loop – Boolean value indicating if the loopback address should be included for traffic.

  • timeout – The socket timeout to assign to the socket

  • apple_p2p – A boolean value indicating if the socket option for Apple Peer-2-Peer should be set.
