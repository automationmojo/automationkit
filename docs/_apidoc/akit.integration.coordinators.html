class akit.integration.coordinators.coordinatorbase.CoordinatorBase(*_args, **_kwargs)[source]¶

Bases: object

The CoordinatorBase utilizes the expected device declarations of type such as ‘network/upnp’ to establish and maintain connectivity and interoperability with a class of devices. A derived coordinator will scan the medium such as a network for the devices declared in the landscape description. The coordinator will also create the threads necessary to maintain communicates with the external devices over the medium.

verify_connectivity(cmd: str = "echo 'It Works'", user: Optional[str] = None, raiseerror: bool = True)[source]¶

Loops through the nodes in the SSH pool and utilizes the credentials for the specified user in order to verify connectivity with the remote node.

Parameters

  • cmd – A command to run on the remote machine in order to verify that ssh connectivity can be establish.

  • user – The name of the user credentials to use for connectivity. If the ‘user’ parameter is not provided, then the credentials of the default or priviledged user will be used.

  • raiseerror – A boolean value indicating if this API should raise an Exception on failure.

Returns

A list of errors encountered when verifying connectivity with the devices managed or watched by the coordinator.
