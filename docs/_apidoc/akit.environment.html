class akit.environment.context.Context(*args, **kwargs)[source]¶

Bases: object

The Context object is a special dictionary derivative that utilizes a ‘path’ style syntax to store and retrieve values and objects. The Context also provides a storage facility that can be replicated or sharded across a distributed environment.
